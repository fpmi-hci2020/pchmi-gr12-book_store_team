## Technologies
Web version is created with:
* Ruby: 2.6.5
* Rails: 6.0.3
* PostgreSQL: 13.0
	
## Setup
To run this project, clone this repo and install dependencies OR use Docker:

```
$ bundle install
$ rails db:setup
$ rails s
```
